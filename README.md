🚀 Islana Carvalho | Portfolio

Bem-vindo(a) ao repositório do meu portfólio pessoal. Este projeto foi desenvolvido para centralizar minha trajetória profissional, destacando minhas competências em Qualidade de Software (QA), Automação de Testes e Desenvolvimento Backend.
💻 Sobre o Projeto

O portfólio é uma aplicação web responsiva que apresenta de forma interativa meus projetos, tecnologias dominadas e experiências. O objetivo é oferecer uma interface limpa e eficiente para recrutadores e desenvolvedores interessados no meu trabalho.
✨ Funcionalidades:

    Apresentação Profissional: Resumo sobre minha atuação técnica.

    Showcase de Projetos: Galeria com links diretos para repositórios e demos.

    Experiência e Educação: Linha do tempo detalhada.

    Interatividade: Filtros de projetos e animações fluidas.

🛠️ Tecnologias Utilizadas

Para a construção desta interface, foquei em uma stack leve e performática:

    HTML5: Estruturação semântica para SEO e acessibilidade.

    CSS3: Estilização moderna com Flexbox, Grid e variáveis para fácil manutenção.

    JavaScript (ES6+): Lógica de interatividade, manipulação de DOM e efeitos de animação.

📂 Estrutura de Arquivos
Bash

├── index.html    # Ponto de entrada e estrutura semântica da página
├── style.css     # Definições de design, responsividade e temas
└── script.js     # Lógica de interações, scroll suave e animações
