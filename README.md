# Portfolio

Portfolio pessoal de Islana Carvalho, com foco em QA, automacao de testes e backend.

## Estrutura

- `index.html`: pagina principal
- `style.css`: estilos da interface
- `script.js`: animacoes e interacoes

## Publicacao no GitHub Pages

1. Crie ou use um repositorio no GitHub.
2. Envie estes arquivos para a raiz do repositorio.
3. No GitHub, acesse `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/ (root)`.
6. Salve e aguarde a publicacao.

Se o repositorio for `islanabeatriz.github.io`, o site sera publicado em:

`https://islanabeatriz.github.io/`

Se o repositorio tiver outro nome, o site sera publicado em:

`https://islanabeatriz.github.io/NOME-DO-REPOSITORIO/`
